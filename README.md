# AnimeVerse
Designed and developed a responsive frontend for an anime streaming portal called "AnimeVerse." The project focuses on creating an immersive, dark-themed user interface that works seamlessly across desktop and mobile devices. 

Key Features:
Responsive Design: Utilized Bootstrap 5 grid system to ensure perfect layout on all screen sizes.
Interactive Navigation: implemented a mobile-friendly navbar with an offcanvas sidebar menu.
Dynamic Visuals: Built a high-resolution image carousel and a unified card gallery for "Recently Added" content.
User Interface: Designed a polished dark mode theme with custom pagination and footer navigation.

Tech Stack:
HTML5
CSS3
Bootstrap 5 (Components & Grid)
JavaScript (Bootstrap Bundle)
